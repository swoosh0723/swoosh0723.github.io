프로젝트 상세 설명

사용자 환경에 맞춰 최적화되는 반응형웹

다양한 디바이스와 해상도에 맞게 최적화되는 반응형웹으로 개발하여
여러 환경의 사용자에게 동일한 사용자경험을 제공하며, 사이트의 유지관리 효율성을 높였고
검색엔진에 친화적이게 제작되었습니다.

  • 웹 접근성

    - 메뉴 호버시 색상이 변경되도록 구현하였습니다. - 컨텐츠 호버시 이미지가 확대되도록 구현하였습니다. - 모든 버튼 호버시 색상, 크기가 변경되도록 구현하였습니다.

  • 크로스 브라우징

    - 이용자가 선호하는 브라우저에서 동일한 화면을 보실 수 있도록 구현하였습니다. - IE9,8의 문제점(이미지 border 생성, 폰트사이즈,여백값등)을 조건 주석문으로 해결 하였습니다.

  • 페이스 오프

    - 미국, 싱가포르에서 런칭 된 글로벌 서비스입니다. - 한국형에 맞게 전반적인 디자인이 변경되었습니다. - 디자이너와함께 디자인 회의를 하였고 맞게끔 퍼블리싱 진행하였습니다.

  • 코드 해석 후 수정 보완

    - 기존에 사용하고 있는 코드를 해석 후 수정 보완 하였습니다.
