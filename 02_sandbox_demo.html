Sandbox Demo 프로젝트

  • Technology

    React ,HTML, SCSS

  • DATE

    2019.02~ 2019.03

  • ROLE

    HTML, SCSS

- 정부에서 진행하는 Sandbox에 제출하기 위해 진행된 프로젝트입니다.
- 부동산을 토크나이즈하여 자산화, 주식화 하는 서비스 입니다.
- 웹표준 및 웹 접근성 적용
- 오픈그래프 설정
- 반응형, 크로스브라우징 적용
- 디자이너와 함께 ux/ui를 개선
- SEO 최적화

프로젝트 상세 설명

사용자 환경에 맞춰 최적화되는 반응형웹

다양한 디바이스와 해상도에 맞게 최적화되는 반응형웹으로 개발하여
여러 환경의 사용자에게 동일한 사용자경험을 제공하며, 사이트의 유지관리 효율성을 높였고
검색엔진에 친화적이게 제작되었습니다.

  • 웹 접근성

    - 메뉴 호버시 색상이 변경되도록 구현하였습니다. - 100%로 들어오는 메인페이지에 하단 스크롤을 유도하기 위해 스크롤 유도하는 애니메이션을 구현하였습니다. - 컨텐츠 호버시 이미지가 확대되도록 구현하였습니다. - 모든 버튼 호버시 색상, 크기가 변경되도록 구현하였습니다.

  • 크로스 브라우징

    - 이용자가 선호하는 브라우저에서 동일한 화면을 보실 수 있도록 구현하였습니다. - IE9,8의 문제점(이미지 border 생성, 폰트사이즈,여백값등)을 조건 주석문으로 해결 하였습니다.

  • 오픈그래프

    - 메신저 카카오톡에서 오픈그래프에 사진과 설명이 잘 보이도록 meta태그로 설정하였습니다.
