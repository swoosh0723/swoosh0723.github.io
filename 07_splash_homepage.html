Splash 홈페이지

Splash 홈페이지

  • Technology

    HTML, CSS

  • DATE

    2018.08

  • ROLE

    HTML, CSS

- 소상공인을 위한 결제 서비스 입니다.
- 블루웨일 토큰을 사용하여 런칭한 서비스의 홈페이지를 제작하는 것입니다.
- 웹표준 및 웹 접근성 적용 하였습니다.
- 디자이너와 함께 ui/ux를 개선하였습니다.
- 반응형으로 제작 하였습니다.

프로젝트 상세 설명

사용자 환경에 맞춰 최적화되는 반응형웹

다양한 디바이스와 해상도에 맞게 최적화되는 반응형웹으로 개발하여
여러 환경의 사용자에게 동일한 사용자경험을 제공하며, 사이트의 유지관리 효율성을 높였고
검색엔진에 친화적이게 제작되었습니다.

  • 웹 접근성

    - 쉬운 네비게이션을 구현 하였습니다. - 키보드, 마우스로 해당 컨트롤이 초점을 받았을 때 시각적으로 구별 할 수 있도록 하였습니다. - 저시력, 시각장애인, 노인들도 콘텐츠를 인식하도록 제공하였습니다.

  • 쉬운 ux/ui

    - 유저가 필수 입력내용을 쉽게 보도록 아이콘을 구성하여 제작하였습니다. - 예매성공, 실패 페이지를 실제 표와 비슷하게 css로 구성하였습니다.
